bug fix

在运行过程中出现如下错误：

参照该链接解决了问题，如果Python是基本于Conda安装的，则Conda上的numpy包中的mkl很容易与系统内库发生冲突，可选择update numpy package in Conda或者设置为系统库。

解决方案:降低numpy的版本，此处笔者将版本降低到1.17.3

pip install numpy==1.17.3
